CALENDAR is a calendar function for OS/2 Presentation Manager. It will display 
the calendar initially for the current month, and for any month and year
combination selected between years 1901 and 2099.                   


CALENDAR is invoked without parameters from either the command line or, more
likely, from within the STARTUP.CMD file. Once active, the month and year for 
display can be can be modified by pressing the buttons for the preceding or
following month. The range of display is January 1901 through December 2099.                     

Sample STARTUP.CMD contents:

     START CALENDAR
     EXIT

(The above assumes that the CALENDAR.EXE file will be found in a directory
listed in the current PATH assignment.)
